# КОПИРАЙТЕР

Этот каталог хранит рабочий контекст агента `@копирайтер`.

## Структура
- `_КОНТЕКСТ/примеры/` - успешные тексты и кейсы для ориентиров
- `_ЛОГИ/` - история задач, выводы и уроки
- `README.md` - краткие правила и назначение

## Обязательный контекст перед работой
Перед первым черновиком агент читает:
1. `.cursorrules`
2. `PROFILE_FOR_AI.md`
3. `03_X/01_CM/01_Работа_CM.md`
4. `03_X/Контекст_X.md`
5. `03_X/02_Twitter/02_Идеи_для_постов.md`

## Как использовать
1. Запустите команду `@копирайтер`.
2. Передайте тему, формат и цель текста.
3. Агент учитывает обязательный контекст и примеры из `_КОНТЕКСТ/примеры/`.
4. После результата дайте оценку, чтобы улучшать базу примеров и логи.

## Рекомендации по наполнению
- В `примеры/` хранить только тексты с оценкой "Отлично"
- В `_ЛОГИ/` добавлять дату, задачу, что сработало, что улучшить
