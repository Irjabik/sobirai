# Агент file-cleaner — база знаний

Здесь лежит **контекст и история** для субагента «уборщик файлов» (определение: `.cursor/agents/file-cleaner.md`).

## Структура

| Путь | Назначение |
|------|------------|
| `_КОНТЕКСТ/примеры/` | Успешные кейсы: как была устроена папка до и после, шаблоны имён, типовое дерево каталогов |
| `_ЛОГИ/` | Уроки и хронология: что сработало, какие ошибки избегать, даты правок |

## Как вести записи

- **Примеры**: один файл на кейс, имя вроде `2026-04-14-inbox-cleanup.md`; внутри — область, правила имён, дерево «до/после», примечания.
- **Логи**: короткие заметки с датой в начале файла или имени; без дубликатов с примерами (ссылка на пример достаточна).

При добавлении материалов придерживайся конвенций самого репозитория (префиксы папок, язык имён файлов).
