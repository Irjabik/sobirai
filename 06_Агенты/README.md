# 07_Агенты

Здесь хранится все, что связано с ИИ-агентами: описания, инструкции, контекст, логи и вспомогательные материалы.

## Что сюда класть

- Описания агентов и их роли (что делает, где применяется).
- Контекст и правила работы для конкретных агентов.
- Промпты, шаблоны задач и рабочие инструкции.
- Технические заметки по настройке и запуску.

## Текущие материалы

- `Копирайтер/` — рабочая папка агента `@копирайтер`.
- `Поиск_работы/` — рабочая папка агента `@поиск_работы`.
- `.cursor/rules/копирайтер.mdc` — правило агента `@копирайтер`.
- `.cursor/rules/поиск_работы.mdc` — правило агента `@поиск_работы`.

## Рекомендуемая структура

Если агентов станет больше, удобно держать формат:

- `01_<agent_name>/` — отдельная папка под каждого агента
- `02_<agent_name>/`
- `...`

Внутри каждой папки: `README.md`, `_КОНТЕКСТ/`, `_ЛОГИ/`, `prompts/` (по необходимости).
