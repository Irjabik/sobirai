# ПОИСК РАБОТЫ

Этот каталог хранит рабочий контекст агента `@поиск_работы`.

## Назначение
- Подбирать релевантные вакансии под цели и профиль Георгия.
- Выдавать шортлист с приоритизацией и рисками.
- Готовить черновики откликов и план следующих шагов.

## Структура
- `_КОНТЕКСТ/` - профиль кандидата, критерии вакансий и шаблоны
- `_ЛОГИ/` - история запусков и результаты по вакансиям
- `prompts/` - готовые формулировки запросов для быстрых запусков
- `lessons.md` - ошибки и правила, которые нельзя повторять

## Обязательный контекст перед работой
Перед первым содержательным ответом агент читает:
1. `.cursorrules`
2. `PROFILE_FOR_AI.md`
3. Весь каталог `06_Агенты/Поиск_работы/` рекурсивно
4. Релевантные файлы из `01_Работа/` и `03_X/` (если нужны для задачи)

## Формат результата
1. Главный вывод (1-2 предложения)
2. Топ вакансий с обоснованием
3. Черновик отклика под лучшую вакансию
4. Следующие шаги (3-5 действий)

## Критерии качества
- Нет выдуманных фактов о кандидате.
- Есть явное сравнение "требование -> соответствие".
- Есть предупреждения по рискам и пробелам.
- Каждая рекомендация завершается конкретным действием.
