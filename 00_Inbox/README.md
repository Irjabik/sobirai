# Inbox

Сюда складывай файлы и заметки без готовой категории: потом разложи по папкам проекта или попроси ИИ предложить, куда что перенести.

## Как пользоваться

- Быстрый сброс: скрины, черновики, ссылки в одном markdown-файле.
- Раз в неделю (или по накоплению): разобрать содержимое в `03_Учеба`, `04_X` (внутри также `05_CM` по Web3 CM/BD), `05_Здоровье`, `06_Личное` или в рабочие папки (`01_Работа`, `02_Сайты`; материалы по Cursor — в `03_Учеба/01_Обучение_Cursor`).

## ИИ

Можешь написать: «разложи файлы из `00_Inbox` по структуре проекта» и прикрепить список или содержимое.
