# Загрузка на Bothost (ручной шаг)

Папка `data/` в репозитории **не коммитится** (см. `.gitignore`). Сессию Telethon держите локально или загрузите через панель хостинга.

1. В панели Bothost откройте «Файловый менеджер» у бота Sobirai.
2. Создайте каталог `data`, если его нет.
3. Загрузите файл `telethon_session.session` в каталог `data` на сервере  
   (имя должно совпадать с `TELETHON_SESSION` в env, например `./data/telethon_session`).
4. Перезапустите бота.
5. В логах ожидается успешная авторизация Telethon и работа коллектора.

Загрузку в аккаунт Bothost выполняете вы в браузере; в Git сессию не кладите.
