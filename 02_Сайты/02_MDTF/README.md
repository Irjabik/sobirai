# 02_MDTF

Материалы по альбому Kanye West *My Beautiful Dark Twisted Fantasy* (рабочее имя папки: MDTF).

## Содержимое

- `Текст` — исходные тексты и подсказки по верстке (SEO, структура).
- `index.html` — главная: обложка, история, YouTube (Runaway), треклист со встроенным Spotify и ссылками на альбом в Spotify и Яндекс Музыке.
- `tour.html` — тур 2010-2011, фон баннера: `img/tour-hero-bg.png` без размытия, в рамке как на сайте.
- Блок фильма: постер `img/runaway-poster.png` + ссылка на YouTube (без iframe). Рамка постера без срезанного угла (раньше использовался декоративный `clip-path` в стиле сайта).
- Секция «Кто стоял за звуком»: портреты Jay-Z, Rihanna, Elton John — `img/credits-jay-z.png`, `credits-rihanna.png`, `credits-elton-john.png`.
- `poll.html` — один вопрос про прослушивание MDTF после сайта (форма только в браузере).
- `site.css` — общие стили для обеих страниц.

Открой `index.html` или `tour.html` в браузере (двойной клик или Live Server). Верстка заточена под телефоны: safe-area, горизонтальный скролл таблицы тура, полоса прогресса скролла и появление блоков при прокрутке (`site-anim.js` + `site.css`). Для Spotify, Яндекс Музыки (внешние ссылки) и YouTube нужен интернет.
