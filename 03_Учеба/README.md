# Учеба

Университет, курсы, самообразование по предметам. Материалы про работу с редактором Cursor и ИИ лежат в подпапке `01_Обучение_Cursor`.

## Что сюда класть

- Программы курсов, силлабусы, конспекты.
- Задания, черновики эссе, списки литературы.
- Дедлайны и планы сессии (таблицы, markdown).

## Чем ИИ может помочь

- Разобрать тему, план эссе, вопросы к экзамену.
- Проверить логику аргументации (не подменять академическую честность: указывай, если нужна только структура, без готового текста на сдачу, если так требует вуз).

## Связь с профилем

Направление: мировая экономика и международный бизнес (УрФУ). В запросах к ИИ можно напоминать предмет и уровень (1 курс).

## Подпапки

- `01_Обучение_Cursor` — онбординг, контекст и практика по Cursor.
- По желанию добавлять, например: `02_Текущий_семестр`, `03_Эссе`, `04_Экзамены` (сохраняй сквозную нумерацию внутри `03_Учеба`).
