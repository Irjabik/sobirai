# 03_Учеба

Здесь хранятся материалы для обучения: конспекты, курсы, практические задания и полезные шпаргалки.

## Что сюда класть

- Конспекты по темам (например: Cursor, Python, маркетинг, коммуникации).
- Полезные ссылки и краткие выжимки из уроков.
- Домашние задания, упражнения и чек-листы по навыкам.
- Личные заметки: что сработало, что нужно повторить.

## Рекомендуемая структура

Можно постепенно добавить подпапки, например:

- `01_Обучение_Cursor`
- `02_Технические_навыки`
- `03_Софт_скиллы`
- `04_Конспекты_курсов`

## ИИ

Можно просить ИИ: собрать план обучения на месяц, разложить заметки по темам, сделать выжимку и карточки повторения.
