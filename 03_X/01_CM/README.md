# 05_CM — Web3: Collab Manager и Partnership Manager (X)

Папка под работу в X как **CM** и рост в **BD**: Hot Intro, Дискорд-тикеты, Alphabot/раффлы, нетворк с КМами, шаблоны ЛС и заметки по проектам.

## Статус (обновляй)

**100+** в X, база WL / раффлы / альфа-группы / Alphabot понятны. Фокус: **Circle of Trust** с КМами и **первые проекты за фикс** (upward от ~$50/нед).

## Краткая выжимка базы

| Блок | Суть |
|------|------|
| Аккаунт | VPN, 2FA, европейский номер, **X Premium**, адекватный визуал и био без деген-клише |
| Рост в ленте | Прогрев перед постом, осмысленные комменты с вопросом в конце, упор на **Articles**, не треды; **приватные Lists 20–30 КМов** |
| CM | Меняем аудиторию через **WL**, **Alphabot** для раффлов; аутрич в **Дискорд** к ранним проектам (ориентир **до ~500–1k** фолловеров), **Hot Intro**; оплата **upfront**, без длинных бесплатных тестов |
| BD | Партнерства, утилиты, токены, AMA/статьи; в ЛС — идея, **метрики**, как сделал, кто ты, CV, без воды; вилка **~$1.5k–5k+** |
| Еще | Альфа-хантер (% с добычи), OTC WL, KOL-менеджмент (% с бюджета), фри-NFT за работу |
